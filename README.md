# Ecommerce Single Product Page 

Simple ecommerce single product page. Can add the product to the cart, switch image by clicking on the thumbnails, or by using the arrow on smaller screen.
Also can open a lighbox when clciking on the active image on large screen 

# Hooks

useIsWindowSmall : Simple custom hook to check is the window is smaller than 768 pixels 
useCart : hook also using react context in order to add product to the cart, display the product in the cart dropdown and delete product from the cart
